# ⚡ NebulaForge X - Quick Start (5 Minutes to Live!)

## 🎯 Get Your AI Game Engine Live in 3 Simple Steps

### Step 1: Open Terminal
```bash
cd /workspace  # or wherever you have the NebulaForge X files
```

### Step 2: Login to Vercel (Free Account)
```bash
vercel login
```
This opens your browser - just click "Continue with GitHub/Google/Email"

### Step 3: Deploy to Production
```bash
vercel --prod
```
Say "yes" to all prompts. That's it! 🚀
